data visualization brand guide

This page set some of the ‘rules’ for the publications by GSS.

Fonts and use of text

Century Gothic Bold should be used for the labels and values on the graph. All axes should be labelled and where possible, the values for data points should be indicated on the graph.

Colour codes

If there is no colour disaggregation present in the data, for example in a bar chart in which all bars have the same colour, a single blue colour should be used.
